# 📇 Contact Management

A modern **Contact Management System** built with **TypeScript**, **JavaScript**, **Node.js**, **CSS**, and **Tailwind CSS**. The application provides an intuitive interface for managing contacts efficiently with complete CRUD functionality and a responsive user experience.

---

## ✨ Features

- ➕ Add new contacts
- 📄 View all saved contacts
- ✏️ Edit existing contacts
- 🗑️ Delete contacts
- 🔍 Search contacts instantly
- 📱 Responsive UI
- 🎨 Modern design with Tailwind CSS
- ⚡ Fast and lightweight interface

---

## 🚀 Getting Started

### Prerequisites

Make sure you have installed:

- Node.js (v18 or later)
- npm

---

### Clone the Repository

```bash
git clone https://github.com/pandeyAsim/Contact-Management.git
```

Move into the project folder.

```bash
cd Contact-Management
```

---

### Install Dependencies

```bash
npm install
```

---

### Run the Application

For development:

```bash
npm run dev
```

or

```bash
npm start
```

depending on your project configuration.
